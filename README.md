Spotify UI Clone 🎧

A Spotify-inspired web player UI built using HTML and CSS. This project replicates the layout and design of the Spotify web application, including sidebar navigation, music cards, and player controls.

  Features

- Sidebar navigation (Home, Search, Library)
- Sticky top navigation bar
- Music cards (Recently Played, Trending, Featured)
- Music player UI (playback controls & progress bar)
- Clean and modern dark theme
- Responsive layout
